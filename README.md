# CSS Ordering Structure

1. Layout / Flow

   - display, position, flex, grid, align-_, justify-_

2. Box Model

   - margin, padding, border, border-radius

3. Visual Appearance

   - background, color, box-shadow, opacity

4. Typography

   - font-size, font-weight, line-height, text-align

5. Interaction / Motion

   - cursor, transition, transform, animation

6. Misc / Overrides
   - overflow, z-index, vendor-specific
